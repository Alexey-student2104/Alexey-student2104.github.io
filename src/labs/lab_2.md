# Отчет по лабораторной работе

## Описание задачи

Цель лабораторной работы состояла в изучении базовых возможностей библиотек **NumPy**, **Pandas**, **Matplotlib** и **Seaborn** для обработки и анализа данных.

В рамках работы требовалось:

- создать и преобразовать векторы и матрицы;
- выполнить основные векторные и матричные операции;
- загрузить данные из CSV-файла;
- провести статистический анализ;
- выполнить нормализацию данных;
- построить графики для визуализации результатов.

В качестве исходных данных использовался файл `students_scores.csv`, содержащий оценки 10 студентов по трем предметам:

- математика  
- физика  
- информатика  

---

## Как была решена задача

Решение было реализовано в файле `main.py` и разделено на несколько этапов.

### 1. Работа с NumPy

Сначала были реализованы базовые операции с массивами:

- создан вектор от `0` до `9`;
- преобразован в матрицу `2x5`;
- создана случайная матрица `5x5`;
- выполнено транспонирование.

### 2. Векторные операции

Реализованы:

- сложение векторов;
- умножение на число;
- поэлементное умножение;
- скалярное произведение.

Все операции выполнены средствами NumPy.

### 3. Матричные вычисления

Используя `numpy.linalg`, реализованы:

- умножение матриц;
- вычисление определителя;
- нахождение обратной матрицы;
- решение системы уравнений `Ax = b`.

### 4. Работа с данными (Pandas)

- CSV загружается через Pandas  
- преобразуется в NumPy массив  

Рассчитаны статистики:

- среднее: **81.5**
- медиана: **84.5**
- минимум: **60**
- максимум: **95**
- стандартное отклонение
- перцентили (25% и 75%)

### 5. Нормализация

Использована формула:

```
(data - min) / (max - min)
```

Значения приводятся к диапазону `[0, 1]`.

### 6. Визуализация

Построены:

- гистограмма распределения оценок;
- тепловая карта корреляции;
- линейный график зависимости оценки по математике от номера студента.

Корреляция между математикой и физикой ≈ **0.96**, что говорит о сильной связи.
