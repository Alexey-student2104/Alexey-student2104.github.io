    Отчет по лабораторной работе

    Ссылка на доску: colab

    Предсказание стоимости недвижимости

    Цель

    Целью лабораторной работы является построение моделей регрессии для предсказания стоимости недвижимости на основе различных характеристик объектов, а также сравнение качества различных алгоритмов машинного обучения.

    Задание

    В рамках работы необходимо: - загрузить и подготовить данные; - разделить признаки и целевую переменную; - обучить несколько моделей регрессии; - оценить их качество с использованием метрик ошибок; - сравнить модели и сделать выводы.

    Целевая переменная: Целевая.Цена

    Ход работы

    1. Загрузка и подготовка данных

    Данные были загружены из файлов: - predict_house_price_training_data.xlsx - predict_house_price_test_data.xlsx

    С использованием библиотеки pandas данные были разделены на: - признаки (features) - целевую переменную (target)

    2. Метрики качества

    Для оценки моделей использовались следующие метрики:

    • MAE (средняя абсолютная ошибка)
    • MSE (среднеквадратичная ошибка)
    • RMSE (корень из MSE)

    RMSE использовалась как основная метрика сравнения моделей.

    Самостоятельная работа

    1. Базовые модели

    Были обучены две базовые модели:

    • Linear Regression
    • Random Forest

    Результаты:

    Модель RMSE
    Linear Regression 201883.24
    Random Forest 135314.02

    Вывод:
    Модель случайного леса значительно превосходит линейную регрессию, что говорит о нелинейной зависимости между признаками и целевой переменной.

    2. Удаление слабых признаков

    Были удалены следующие признаки:

    • Год реновации
    • Количество этажей
    • Спальни
    • Состояние
    • Площадь подвала
    • Ванные

    После этого модель Random Forest была обучена повторно.

    Результат:

    • RMSE = 133286.12

    Вывод:
    Удаление слабых признаков немного улучшило качество модели.

    3. Подбор параметров Random Forest

    Были подобраны параметры модели:

    • n_estimators = 300
    • min_samples_leaf = 2
    • max_features = 0.7

    Результат:

    • RMSE = 134561.94

    Вывод:
    Подбор параметров также улучшил модель, однако эффект оказался незначительным.

    4. Обучение других моделей

    Были протестированы различные модели:

    • DecisionTreeRegressor
    • LassoCV
    • RidgeCV
    • ElasticNetCV
    • MLPRegressor
    • GradientBoostingRegressor
    • HistGradientBoostingRegressor

    Все модели были обучены и оценены по метрикам.

    Лучший результат:

    Модель RMSE
    HistGradientBoostingRegressor 121654.97

    Общий вывод

    В ходе лабораторной работы были построены и сравнены различные модели регрессии для предсказания стоимости недвижимости.

    Основные выводы:

    • Линейные модели показывают худшие результаты из-за сложности и нелинейности данных;
    • Ансамблевые методы (Random Forest, Gradient Boosting) значительно превосходят базовые модели;
    • Удаление слабых признаков и настройка параметров дают небольшое улучшение;
    • Лучшей моделью оказался HistGradientBoostingRegressor, показавший минимальную ошибку.

    Таким образом, для данной задачи наиболее эффективными являются ансамблевые методы градиентного бустинга.
